## Overview

This guide walks you through a minimal setup so you can start tracking production.

## Step 1: Create a facility

- Name your facility and add zones or areas.
- Add at least one production line.

## Step 2: Add machines and shifts

- Create machines for each line.
- Assign staffing shifts for accurate analytics.

## Step 3: Define a SKU and BOM

- Create a product SKU.
- Add a bill of materials with required components.

## Step 4: Release a work order

- Create a work order for the SKU.
- Assign a line and planned start date.
- Start the order and move it through stations.

## Optional: Send machine data

- Create an API token scoped to your organization.
- Post part counts to the ingestion API.
