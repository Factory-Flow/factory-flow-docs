---
title: "Release a work order"
description: "Create, schedule, and start your first production order."
---

This step turns your factory model and product setup into a live production run.

## Create the work order

<Steps>
  <Step title="Start a new work order">
    Go to **Work Orders** and click **Create Work Order**.
  </Step>
  <Step title="Choose the SKU and quantity">
    Select the product you defined and enter the target quantity.
  </Step>
</Steps>

## Schedule and assign

<Steps>
  <Step title="Assign a line">
    Pick the production line that will run the order.
  </Step>
  <Step title="Set planned dates">
    Add a planned start date (and end date, if needed) to align with shifts.
  </Step>
</Steps>

## Start and track the order

<Steps>
  <Step title="Release the order">
    Change the status to **In Progress** when production begins.
  </Step>
  <Step title="Move through stations">
    Update status as work progresses to keep reporting accurate.
  </Step>
</Steps>
