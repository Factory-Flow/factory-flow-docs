Layouts provide a visual representation of your factory floor. Upload blueprints, position machines, and create an interactive map of your production environment.

## Understanding layouts

A layout in Factory Flow is a visual floor plan that shows:

- Physical arrangement of machines
- Production flow and material paths
- Equipment positions with real-time status
- Quick navigation to machine details

Layouts help your team:

- **Visualize operations** - See the entire floor at a glance
- **Navigate quickly** - Click machines to access their data
- **Communicate** - Share visual context with stakeholders
- **Plan changes** - Design new arrangements before implementation

## Creating a layout

To create a new layout:

1. Navigate to **Layouts** in the sidebar
2. Click **Create Layout**
3. Configure the layout:

### Layout settings

| Field | Description |
|-------|-------------|
| **Title** | Name for this layout (e.g., "Main Floor - Level 1") |
| **Width** | Canvas width in units |
| **Height** | Canvas height in units |
| **Description** | Optional notes about the layout |
| **Visibility** | Public or private to your organization |

4. Click **Create**

## Uploading blueprints

Add a floor plan image as the background:

1. Open the layout for editing
2. Click **Upload Blueprint**
3. Select an image file (JPG, PNG)
4. The image uploads and displays as the background

### Supported formats

- JPEG/JPG
- PNG
- Recommended resolution: 1920x1080 or higher
- Maximum file size: 10MB

<Tip>
  For best results, use high-resolution architectural drawings or CAD exports. The image will scale to fit the canvas dimensions.
</Tip>

### Updating blueprints

To replace an existing blueprint:

1. Click **Upload Blueprint** again
2. Select the new image
3. The previous image is replaced

### Downloading blueprints

To retrieve the current blueprint:

1. Click **Download Blueprint**
2. The image saves to your device

## Positioning machines

Place machines on your layout to create an interactive floor map.

### Adding machines to a layout

1. Open the layout editor
2. Click **Add Machine**
3. Select a machine from your organization
4. Click on the canvas to place it
5. Drag to reposition as needed

### Machine properties on layouts

Each machine on a layout has:

| Property | Description |
|----------|-------------|
| **Position X** | Horizontal position on canvas |
| **Position Y** | Vertical position on canvas |
| **Width** | Display width of machine icon |
| **Height** | Display height of machine icon |
| **Z-Index** | Layer order for overlapping machines |

### Editing machine positions

To move a machine:
1. Click and drag it to a new position
2. Release to set the new location

To resize:
1. Select the machine
2. Drag the corner handles
3. Release to set the new size

To adjust layering:
1. Select the machine
2. Change the Z-Index value
3. Higher values appear on top

### Removing machines from layouts

1. Select the machine on the layout
2. Click **Remove from Layout**
3. The machine is removed from the layout but not deleted from your organization

## Layout visualization

### Machine status on layouts

Machines display their current status:

| Status | Visual |
|--------|--------|
| Connected | Green indicator |
| Disconnected | Gray indicator |
| Warning | Yellow indicator |
| Critical | Red indicator |

### Cycle time display

Machines can show real-time cycle time:
- Current cycle time value
- Color coded based on thresholds
- Updates automatically

### Click to navigate

Click any machine on the layout to:
- View quick stats
- Navigate to machine detail page
- Access live data

## Managing layouts

### Viewing layouts

The Layouts page shows a gallery of all layouts:

- Layout thumbnail (if blueprint uploaded)
- Title and description
- Machine count
- Last updated date

### Editing layouts

1. Click on a layout to open it
2. Click **Edit** to enter edit mode
3. Make changes to positions, machines, or settings
4. Click **Save** to apply changes

### Deleting layouts

1. Open the layout
2. Click **Delete**
3. Confirm the deletion

<Warning>
  Deleting a layout does not delete the machines on it. The machines remain in your organization and can be added to other layouts.
</Warning>

## Layout visibility

Control who can see your layouts:

### Private layouts

- Visible only to organization members
- Default setting for new layouts
- Requires authentication to view

### Public layouts

- Can be shared via link
- Useful for stakeholder presentations
- Still requires organization access for editing

To change visibility:
1. Open the layout
2. Click **Edit**
3. Toggle the **Visibility** setting
4. Click **Save**

## Best practices

### Naming conventions

| Good | Why |
|------|-----|
| "Building A - Floor 1" | Clear location hierarchy |
| "Assembly Line Overview" | Describes what's shown |
| "Q1 2024 Proposed Layout" | Indicates purpose and timing |

### Blueprint preparation

1. **Use high resolution** - Ensures readability when zoomed
2. **Include scale reference** - Helps with accurate machine sizing
3. **Remove clutter** - Simplify to show essential features
4. **Consistent orientation** - North up or entry at bottom

### Machine placement

1. **Match physical positions** - Accuracy aids navigation
2. **Use consistent sizing** - Maintain proportions
3. **Leave spacing** - Avoid overlapping machines
4. **Group by function** - Visual organization helps comprehension

### Multiple layouts

Consider creating separate layouts for:
- Different floors or buildings
- Different zoom levels (overview vs. detail)
- Current state vs. planned changes
- Different production areas

- Explain production flow
- Showcase capabilities

Create draft layouts to:
- Test new arrangements
- Evaluate flow changes
