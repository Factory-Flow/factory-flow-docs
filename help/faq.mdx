## Getting started

<AccordionGroup>
  <Accordion title="How do I get started with Factory Flow?">
    Start with the [Quickstart guide](/getting-started/quickstart). It walks you through creating an account, setting up your organization, adding your first machine, and sending production data.
  </Accordion>

  <Accordion title="What do I need before I can start?">
    You need:
    - A valid email address for account creation
    - Basic information about your machines (names, types)
    - A way to send data from your machines (PLC, edge gateway, or custom integration)

    See [System requirements](/getting-started/requirements) for full details.
  </Accordion>

  <Accordion title="How do I invite my team?">
    1. Go to **Organization Settings → Members**
    2. Click **Invite Member**
    3. Enter their email and select a role (Viewer, Editor, or Admin)
    4. They'll receive an email invitation to join

    Learn more in [Account setup](/getting-started/account-setup).
  </Accordion>
</AccordionGroup>

## Machines and data

<AccordionGroup>
  <Accordion title="How do I connect a machine to Factory Flow?">
    1. Create a machine in Factory Flow
    2. Configure the data source (Factory Flow API)
    3. Create an API token with Read/Write permissions
    4. Send data to the machine's ingestion endpoint

    See [Data ingestion](/analytics/ingestion) for detailed instructions.
  </Accordion>

  <Accordion title="What data can I send to Factory Flow?">
    The ingestion API accepts part counts:
    - `part_count_total` - Total parts produced
    - `part_count_good` - Parts passing quality
    - `part_count_bad` - Rejected/scrapped parts

    Factory Flow calculates cycle time, OEE, and other metrics from this data.
  </Accordion>

  <Accordion title="How often should I send data?">
    We recommend sending data every 1-60 seconds, depending on your cycle time. More frequent updates provide better real-time visibility, but the ingestion API is rate-limited to prevent overload.
  </Accordion>

  <Accordion title="My machine shows as disconnected. What should I check?">
    1. Verify the API token is valid and not revoked
    2. Check network connectivity from your machine/gateway
    3. Confirm the machine UUID is correct
    4. Look for errors in your ingestion logs
    5. Check "Last Data Received" on the machine page

    See [Troubleshooting](/help/troubleshooting) for more help.
  </Accordion>
</AccordionGroup>

## Analytics

<AccordionGroup>
  <Accordion title="How is OEE calculated?">
    OEE (Overall Equipment Effectiveness) combines three factors:

    **OEE = Availability × Performance × Quality**

    - **Availability** = Run Time ÷ Planned Production Time
    - **Performance** = (Ideal Cycle Time × Total Count) ÷ Run Time
    - **Quality** = Good Count ÷ Total Count

    Learn more in [Key metrics](/analytics/kpis).
  </Accordion>

  <Accordion title="How is cycle time calculated?">
    Cycle time is calculated from the timestamps of consecutive part count submissions. Factory Flow measures the time between data points to determine how long each production cycle takes.
  </Accordion>

  <Accordion title="Can I export my analytics data?">
    Yes, you can export data to Excel or PDF:
    1. Navigate to **Analytics**
    2. Select a machine and time range
    3. Click **Export**
    4. Choose your format

    See [Exports](/analytics/exports) for details.
  </Accordion>

  <Accordion title="How long is my data retained?">
    Data retention depends on your subscription plan. Contact support or check your plan details in **Organization Settings → Billing** for specifics.
  </Accordion>
</AccordionGroup>

## Organizations and teams

<AccordionGroup>
  <Accordion title="What's the difference between Viewer, Editor, and Admin roles?">
    | Role | Can do |
    |------|--------|
    | **Viewer** | View dashboards, analytics, and reports (read-only) |
    | **Editor** | Create/edit machines, lines, shifts, layouts, and export data |
    | **Admin** | All Editor permissions plus manage members, API tokens, billing, and settings |
  </Accordion>

  <Accordion title="Can I belong to multiple organizations?">
    Yes! You can be a member of multiple organizations and easily switch between them using the organization switcher in the sidebar.
  </Accordion>

  <Accordion title="How do I leave an organization?">
    1. Go to **Organization Settings → Members**
    2. Find your name in the list
    3. Click **Leave Organization**
    4. Confirm your decision

    Note: If you're the last member, leaving will delete the organization.
  </Accordion>

  <Accordion title="How do I delete my organization?">
    Only the last remaining member can delete an organization:
    1. Remove all other members first
    2. Go to **Organization Settings**
    3. Scroll to **Danger Zone**
    4. Click **Delete Organization**
    5. Type the organization name to confirm

    This action is permanent and cannot be undone.
  </Accordion>
</AccordionGroup>

## Authentication and security

<AccordionGroup>
  <Accordion title="What sign-in methods are supported?">
    Factory Flow supports:
    - Email (passwordless magic links)
    - Google (Workspace and personal accounts)
    - Microsoft (Entra ID/Azure AD)
    - GitHub

    See [SSO](/integrations/sso) for details.
  </Accordion>

  <Accordion title="I can't sign in. What should I do?">
    1. Check you're using the correct email address
    2. Try a different sign-in method if available
    3. Clear your browser cache and cookies
    4. Try an incognito/private browser window
    5. Contact [support@factoryflow.com](mailto:support@factoryflow.com)
  </Accordion>

  <Accordion title="How do I reset my password?">
    If using email authentication, click "Forgot password" on the login page. You'll receive a link to reset your password.

    If using SSO (Google, Microsoft, GitHub), manage your password through that provider.
  </Accordion>
</AccordionGroup>

## API and integrations

<AccordionGroup>
  <Accordion title="Where do I find my API token?">
    1. Go to **Organization Settings → API Tokens**
    2. Create a new token or view existing ones

    Note: Tokens are only displayed once when created. If you've lost a token, create a new one.
  </Accordion>

  <Accordion title="What's the API base URL?">
    ```
    https://app.factoryflow.com/api
    ```

    For machine data ingestion:
    ```
    POST https://app.factoryflow.com/api/machines/{machine_uuid}/parts
    ```
  </Accordion>

  <Accordion title="Are there API rate limits?">
    Yes, API requests are rate-limited per organization and per machine. If you receive a 429 error, wait and retry. See [API authentication](/api/authentication) for details on handling rate limits.
  </Accordion>
</AccordionGroup>

## Billing

<AccordionGroup>
  <Accordion title="How do I update my payment method?">
    1. Go to **Organization Settings → Billing**
    2. Click **Manage Billing** to access the Stripe portal
    3. Update your payment method

    See [Billing](/administration/billing) for details.
  </Accordion>

  <Accordion title="How do I cancel my subscription?">
    1. Go to **Organization Settings → Billing**
    2. Click **Manage Billing**
    3. Click **Cancel subscription** in the Stripe portal

    Access continues until the end of your billing period.
  </Accordion>

  <Accordion title="How do I download invoices?">
    1. Go to **Organization Settings → Billing**
    2. Click **Manage Billing**
    3. Navigate to **Invoices** in the Stripe portal
    4. Download PDFs for any invoice
  </Accordion>
</AccordionGroup>
